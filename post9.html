A/B Testing

Content Experiment with Google Optimize

Posted by Eilis Lee on June 7, 2020

Google Optimize offers tools for site personalization and testing solutions. It's built on top of Google Analytics and allows you to test a hypotheses for site changes without the need to publish any real development changes.

Google Optimize has a number of testing options but for this experiment, A/B Testing was set up to test between two versions of this site's homepage: the original and the variant.

Content Changes

Targeting and Variant

Creating an Experience with Google Optimize includes three steps:

  1. Make the visual site changes
  2. Choose a target audience
  3. Set a testing objective

Last week I set up a click event goal with Google Analytics on the blog post titled Google Analytics. For this week's experiment, my goal was to guide more people to that post by making it stand out more.

The A/B test experiment was created with the following information:

Required Information Changes/Data
Variant Increase font size
Target audience Users in the city of Kent
Test Objective Pageviews

I increased the size of the font and chose to have this version of the homepage displayed only to users in the city of Kent. The pageviews objective is based on Google Analytics data.

Original font size
Increased font size

Findings

Usability Testing

After setting up the experience, I had a few people from Kent visit my site for an informal usability test.

Task Findings
Find an interesting article to read from the homepage All participants clicked on the targeted Google Analytics post

Given the task of finding an interesting article to read, each of the participants clicked on the Google Analytics blog post, which was surprising to me as my inital thought was that the participants might actually look for an article that they were interested in. Several of them admitted that they chose the Google Analytics post because of it's large font size.

I ran the Google Optimize test for a few days but the reporting on the experience did not offer a clear leader in the experiment. I may need more visitors to improve the reporting.

Final Thoughts

I can see the value in Google Optimize's testing tools and found the whole process to be really user-friendly. With that small change in font size, I was able to acheive the goal of guiding more people towards the target post. For the sake of style consistency though, I wouldn't actually make this change to the site. The change would be to the actual content of the title.
