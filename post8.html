Google Analytics Goals and Events

Adding Goals and Events

Posted by Eilis Lee on May 31, 2020

Google Analytics allows administrators to set up and use a feature called "Goals" to track each time a user completes a specific action while on their website. This could include many different actions and events such as signing up for a newsletter or completing a purchase.

Goals and Events

Google Analytics has a number of templates that administrators can use to get started as well as the option to create custom events and goals, which is what I setup earlier this week.

Google Analytics Goals and Events

Setting up custom goals, there are four types to choose from: destination, duration, pages/screens per session, and events.

Google Analytics Custom Goals

I chose to setup three separate goals:

  1. Event: click on Google Analytics blog post from the homepage
  2. Pages/Screens per session: visit more than 1 page per session
  3. Duration: time spent greater than 30 seconds per session

Google Analytics Goals and Events

After a couple days of collecting data, I was surprised to see that only two of the three goals were actually met.

Google Analytics Goals Overview

I did a little research and realized that the event goal I had set up was missing a tag in the JavaScript code. This was corrected today and verified in realtime so I'm hoping to see some data on it tomorrow.
